# Food Safety Inspection Tracker

## Description
This project is an ASP.NET Core MVC application used to track food safety inspections for local councils. 
---

## different frameworks
- ASP.NET Core MVC  
- Entity Framework Core  
- SQLite / SQL Server  
- Identity
- Serilog   
- xUnit

---

## Features
- manage
- Role-based access 
- Dashboard 
- Logging with Serilog 
- Global error handling  

---

## Roles
- **Admin** → Full access  
- **Inspector** → Create inspections and follow-ups  
- **Viewer** → Read-only access
